# Arcline Dashboard

## Run the app

```bash
pip install flask
python app.py
```

Then open: http://localhost:5000

## Features
- Client dashboard with stats, chart, system status, call log
- Receptionist toggle with confirmation modal
- Payments page with invoice history
- Admin panel: manage clients, reset passwords, update invoice status
- Add new clients
- Delete clients
- SQLite database (auto-created on first run)
